This is my personal CV website, built to showcase my skills, experience, and projects. It serves as a digital resume, allowing potential employers and collaborators to learn more about me and my professional background.

Languages Used
HTML5 CSS3 JavaScript
Features
Clean, responsive design that adapts to different screen sizes.
Sections for about me, work experience, skills, and projects.
Social media and contact links.
Option to download my resume in PDF format.
